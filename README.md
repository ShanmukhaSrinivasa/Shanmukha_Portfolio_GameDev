# SHANMUKHA_OS v1.0.4

A high-performance, data-driven portfolio archive for Game Development.

## Architecture
- **Framework**: React 18 + Vite (optimized build pipeline)
- **Routing**: React Router 6 (dynamic slug-based project generation)
- **Sound Engine**: Custom Web Audio API wrapper for synthesized UI feedback
- **Styling**: Tailwind CSS with a Brutalist/Terminal aesthetic
- **Deployment**: Automated CI/CD via Vercel

## Key Technical Features
- **Data-Driven UI**: Entire project archive is managed via a single TypeScript manifest.
- **Dynamic Embeds**: Automated itch.io WebGL integration for live gameplay execution.
- **Splash Sequence**: Managed React state-based "boot sequence" for terminal immersion.
- **Responsive Systems**: Fully fluid layout optimized for both desktop and mobile review.
